# iOS Starter Apps

A collection of small apps I built while learning Swift, UIKit, Interface Builder, and basic iOS app architecture through an online course.

The repository includes:

- **I Am Rich / I Am Poor** — first experiments with layouts, labels, and image assets
- **Dicee** — a dice roller with randomized state
- **Xylophone** — buttons, audio playback, and simple interaction feedback
- **Egg Timer** — timers, progress updates, and sound
- **Quizzler** — a quiz app organized around a simple model and controller
- **Destini** — a branching story app

## Running a project

Open the `.xcodeproj` file inside the app folder with Xcode, choose an iOS simulator, and run the project. Each folder is independent; there is no shared build step.

These are older learning projects, so newer Xcode versions may prompt you to update project settings or make small compatibility fixes. They are kept as a record of my early iOS work rather than actively maintained apps.
